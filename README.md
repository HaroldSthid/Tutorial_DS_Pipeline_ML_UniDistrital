# Tutorial_DS_Pipeline_ML_UniDistrital

Welcome! 👋

Este proyecto es un **tutorial interactivo** para aprender y aplicar un pipeline completo de Machine Learning para segmentación de clientes, usando Python y librerías populares como pandas, scikit-learn, matplotlib y más.

## 🚀 Quick Access / Acceso Rápido

👉 **[Ver el tutorial online (GitHub Pages)](https://haroldsthid.github.io/Tutorial_DS_Pipeline_ML_UniDistrital/)**

## ¿Qué encontrarás aquí? / What will you find?
- Explicación paso a paso del pipeline de ML (EDA, limpieza, clustering, clasificación, reglas de negocio, hacks de Colab, etc.)
- Código listo para copiar y usar en Google Colab o Jupyter
- Tips, hacks y mejores prácticas para análisis de datos
- Ejemplo de segmentación de clientes con K-Means y árboles de decisión

## 📦 Tecnologías / Technologies
- Python 3.x
- pandas, numpy, scikit-learn, matplotlib, seaborn, plotly
- Google Colab (opcional)
- HTML5 + CSS3 (para la versión web)

## Cómo usarlo / How to use
1. Accede a la versión web desde el link arriba ☝️
2. Copia los snippets de código y pégalos en tu notebook de Colab/Jupyter
3. Sigue los pasos y personaliza el pipeline para tus propios datos
